密码哈希(Password Hashing)
  • July 28, 2019
  • security  

密码哈希（password hashing），是常用的密码管理技术。在 web 开发等工程实践中，账户密码等敏感信息通常不会明文存储到数据库，而是转换成一段哈希值存储。这样即使哪天数据库泄露了，用户的密码也不会一眼被识破。密码哈希有以下特点：

  1. 把任意长度的字符串，转换成固定长度的随机字符串。
  2. 单向加密，不可逆转
